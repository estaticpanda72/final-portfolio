Website Redesigned

Background


Hello there! Let's talk a bit about the idea behind redesigning Suzanne Collins' website homepage. My goal was to identify a webpage to analyze according to usability and accessibility standards. Then redesign and build the redesigned version of said webpage. Additionally, I analyzed the usability of the web page using the following criteria:

    Usability
    Learnability
    Memorability
In addition to analyzing the website on the three usability guidelines, I use WebAIM WAVE to detect possible accessibility problems.

The Process


I selected this website because it's a simplistic website but I believe it almost plays into that element a bit too much. I think there's a lot of content that's simply just dumped on the pages without much thought given to best ensure users see that information in a manner that works best for them. Important information is essentially not positioned in a manner that's best suited to ensuring a user sees it.

Key things I chose to redesign

Problem Fix
Large blocks of texts Implemented a scrolling slideshow that showcases each book's reviews and awards
Low contrast Updated the font and font color
Menu doesn't stand out Repositioned the menu buttons
Bland aesthetic Repositioned the heading and introduced a more pleasing and cohesive theme

Takeaways


The entire process of this redesign was truly a learning experience. I got a chance to experience the full UIUX design cycle. Working from identifying problems with a current interface, then designing both low and high fidelity mockups to get a sense for what might be improved. I also created a style guide which influenced most of my theme choices: color, font, button style, and so on. Finally, I dove into the actual interface redesign. I throughly felt that this project helped me expand upin my html and css skills as well.
